# Text to speech

Text to speech is a react native application that helps you pronounce words in different languages.

# Demo
[text-to-speechApp](https://snack.expo.io/@akinmyde/speech)

# Prerequisite
* Node 12.0
* Watchman
* Expo cli
* Xcode

# Usage
```npm install```
